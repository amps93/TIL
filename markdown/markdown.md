# Markdown

| 명령어           | 실행 결과                                    |
| ---------------- | -------------------------------------------- |
| `# 제목`         | 제목(# 개수에 따라 크기 변동)                |
| `숫자. `         | 숫자 목록                                    |
| `* `, `+ `, `- ` | 순서없는 목록                                |
| `ctrl /`         | 소스 모드                                    |
| `''`             | 코드 블럭(왼쪽상단 1옆에 있는 버튼)          |
| `<>` , `''' '''` | 코드 블럭(우측꺼는 왼쪽상단 1옆에 있는 버튼) |
